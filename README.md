# 📊 Pandas Data Analysis Practice

This repository contains a structured set of hands-on exercises focused on **data analysis using Pandas**. Each script corresponds to a specific task covering the full data analysis workflow — from data loading to outlier treatment.

---

##  Project Overview

The goal of this repository is to build strong fundamentals in:

* Data loading and inspection
* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Feature engineering
* Outlier detection and treatment

Each exercise is implemented as an independent Python script using real or simulated datasets.

---

##  Project Structure

```
practice_projects/

├── data_loading_inspection.py
├── data_cleaning.py
├── eds.py
├── feature_engineering.py
├── outlier_detection.py
```

---

## 🧠 Exercises Breakdown

### 🔹 Data Loading & Inspection

* Load datasets using Pandas and Seaborn
* Explore structure, data types, and memory usage
* Identify missing values and basic statistics

---

### 🔹 Data Cleaning

* Handle missing values
* Fix invalid data (e.g., incorrect ages)
* Standardize formats (dates, text)
* Remove duplicates and validate fields

---

### 🔹 Exploratory Data Analysis (EDA)

* Analyze relationships between variables
* Group and aggregate data
* Create visualizations:

  * Histograms
  * Box plots
  * Scatter plots
  * Correlation heatmaps

---

### 🔹 Feature Engineering

* Create new meaningful features
* Extract date-based features
* Perform aggregations
* Encode categorical variables
* Apply scaling techniques

---

### 🔹 Outlier Detection & Treatment

* Detect outliers using:

  * Z-score method
  * IQR method
  * Box plots
* Apply treatments:

  * Removing outliers
  * Capping (Winsorization)
  * Log transformation
* Compare effects using visualizations
